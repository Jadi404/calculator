# Quick Maths Calculator
A desktop calculator made with PyQt6 that makes use of a clean Ui, as well as a responsive button layout.

## Features
* Basic arithmetic operations (+,-,x,÷)
* Clean Ui
* Hover state for buttons
* Responsive layout
* Clear functionality
* Error handling

## Installation

1. Clone the repository

```bash
git clone https://github.com/Jadi404/calculator.git
```

2. Navigate to the project folder

```bash
cd calculator
```

3. Install dependencies

```bash
pip install -r requirements.txt
```
